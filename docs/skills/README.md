# Questlog Skills Workspace

This folder is the staging area for manager-facing skill files derived from the
Questlog situation taxonomy.

Each skill should live in its own directory:

```text
docs/skills/<skill-name>/SKILL.md
```

Rules for this workspace:

- one directory per distinct manager situation
- one `SKILL.md` per directory as the primary procedure file
- no duplicated trigger territory across sibling skills
- concrete instructions only
- explicit failure handling
- split rather than bloat

The full skill backlog and naming scheme live in the manager situation taxonomy
plan.
